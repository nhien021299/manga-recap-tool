When implementing a new feature:
1. Ask orchestrator to classify the task.
2. Ask virtual-strip-architect to define types and coordinate flow.
3. Ask the relevant specialist to design the algorithm.
4. Ask image-worker-performance if the feature touches decoding, extraction, or scanning.
5. Ask crop-editor-ux if the feature changes user interactions.
6. Produce:
   - architecture summary
   - implementation plan
   - code changes
   - test cases

Always attach `../project-rules.md` to every specialist step.
